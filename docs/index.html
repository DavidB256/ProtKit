Welcome to ProtKit

ProtKit is a suite of tools for working with .pdb files. It includes the ProtKit Scripting Language (PKSL). It was created by David Bass to assist with the modeling efforts of the 2021 Virginia iGEM team.

Why ProtKit?

ProtKit was created as an open-source tool to grant structural biologists greater control over the files that they work with. It brings a number of utilities together into one user-friendly program. Think of it like data science, but for proteins.
